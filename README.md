**Gestor de Productos – Proyecto Full Stack**
Aplicación web sencilla para gestionar productos con precio y cantidad, desarrollada con React, Axios, Node.js, Express y persistencia en JSON.

**Tecnologías**
Frontend: React + Vite
Backend: Node.js + Express
HTTP Client: Axios
Estado: Hooks (useState, useEffect, useContext, useReducer)
Navegación: React Router
Gestión global: Context API
Base de datos: database.json
Control de versiones: GitHub

**Estructura del Proyecto**
/frontend /backend .gitignore README.md

**Requisitos**
Node.js 18+

**Instalación y Ejecución**
Backend
cd backend npm install node server.js
Backend disponible en:
http://localhost:3000
Frontend
cd frontend npm install npm run dev
Frontend disponible (por defecto) en:
http://localhost:5173
Endpoints del Backend
Método Ruta Descripción
GET /products Lista productos POST /products Crea producto PUT /products/:id Actualiza producto DELETE /products/:id Elimina producto
